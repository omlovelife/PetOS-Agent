不只是循环动画，是会相处的 Agent

多数桌面宠物只会重复动作。PetOS 把宠物做成真正的 Agent：感知你何时专注、何时可以搭话，并把经历沉淀成按宠物隔离的本地记忆，越相处越像「你的那一只」。

  • 记得你们聊过什么 对话与经历按宠物分开保存，不会串台，也不会默认上传。
  • 懂什么时候该安静 专注模式、勿扰时段、话唠程度可调；番茄钟进行时它会收敛存在感。
  • 始终是宠物，不是客服 桌面气泡是唯一聊天入口，始终以角色身份回应，不跳出人设。

桌面宠物系统能做什么

从常驻存在感到自主活动，再到自备模型与本地数据，围绕「一只真正待在桌面上的宠物」展开。

  • 常驻桌面的存在感

    透明置顶、可拖动互动，会自己走走停停。没有突兀的任务栏入口，像贴在桌面上的小伙伴，而不是又一个窗口。

  • 互动有节奏

    点击、拖放、摸摸，会有反应，但不会连闪刷屏。想说话时点开气泡即可。

  • 自主活动的小生命

    不说话时也会走走、停停、发呆。主动搭话有冷却与日上限，陪着你，不吵你。

  • 自备模型，数据在本机

    填入 OpenAI 兼容的 Base URL、Key 与模型名即可对话。密钥与宠物数据默认只存在你的电脑上；联网能力由你显式配置。

  • 专注时更安静

    番茄钟、勿扰时段与话唠程度一起调节存在感。Agent 会感知场景，而不是无脑打扰。

  • 选一只属于你的宠物

    内置默认伙伴，也可从 Petdex 或本地文件夹导入。每只宠物有自己的形象、性格与记忆空间。

三步开始

  1. 01

    安装

    下载 Windows 或 macOS 安装包，装好后打开即可。

  2. 02

    选一只宠物

    使用默认伙伴，或从 Petdex / 本地文件夹导入你喜欢的形象。

  3. 03

    配置自己的模型

    填入 API 后开始对话。密钥只存在本机，PetOS 不提供模型服务本身。

设计原则

  • 本地优先 对话与记忆按宠物隔离，默认不出本机。
  • 角色一致 Agent 始终以宠物身份行动与回应。
  • 安静陪伴 可感知场景，主动搭话有节制。
