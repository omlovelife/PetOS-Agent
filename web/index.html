一只真正待在桌面上的宠物

透明置顶、可拖动互动、会自己走走停停。想说话时点开气泡；不想被打扰时，它会安静地待着。

  • 常驻 无任务栏入口，像贴在桌面上的小生命
  • 互动 点击、拖放、摸摸，有节奏的反应而不是连闪
  • 对话 桌面气泡是唯一聊天入口，始终以宠物身份回应

三步开始

  1. 01

    安装

    下载 Windows 或 macOS 安装包，装好后打开即可。

  2. 02

    选一只宠物

    内置默认伙伴，也可从 Petdex 或本地文件夹导入。

  3. 03

    配置自己的模型

    填入 OpenAI 兼容的 Base URL、Key 与模型名，密钥只存在本机。

陪着你，不吵你

专注模式、勿扰时段、可调的话唠程度。番茄钟进行时它会更安静。主动搭话有冷却与日上限，不会刷屏。

本地优先：对话与记忆按宠物隔离保存在你的电脑上；联网能力由你显式配置。
