它自己会动，也会自己聊

不是循环壁纸。它会在桌面上走动、发呆，并用宠物身份和你对话。

  • 自主运动 透明置顶常驻桌面，走走停停，摸头招手、摸身子开心、摸爪子跳跃。
  • 自主对话 桌面气泡里始终以宠物身份回应，记忆按宠物隔离，越聊越像你的那一只。
  • 番茄钟陪伴 专注模式与勿扰时段可调。番茄钟进行时它会收敛存在感，不打断你。

兼容 Codex 宠物，数据留在本机

支持 Codex / Petdex 宠物规范，可导入更多伙伴。密钥与记忆默认只存本机；更多能力持续开发中。

  • Codex 宠物规范

    内置豆包，也可从 Petdex 或本地文件夹导入符合规范的宠物。

  • 自备模型

    填入 OpenAI 兼容的 Base URL、Key 与模型名即可开始对话。

  • 持续更新

    更多陪伴与互动能力开发中，下载即用当前版本，后续持续进化。
