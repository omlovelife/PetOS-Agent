# PetOS

**成长陪伴的 AI 宠物 Agent。**

一只常驻桌面的小生命：会自主活动、会记住你们的相处、越陪越懂你。对话始终以宠物身份回应；密钥与记忆默认只留在本机。

官网：https://petos.lifemindx.com/

---

## 成长陪伴

多数桌面宠物只会循环动画。PetOS 把宠物做成真正的 AI Agent：它会陪着你活动与说话，把相处写进按宠物隔离的本地记忆，也会在你专注时安静下来。

- **陪伴在侧** - 透明置顶常驻桌面，走走停停、点点摸摸
- **自主成长** - 记忆按宠物分开保存，相处越久越像「你的那一只」
- **安静懂你** - 专注模式、勿扰时段、话唠程度可调

---

## 功能一览

### 常驻陪伴

透明置顶、可拖动互动。没有突兀的任务栏入口，像贴在桌面上的小伙伴。

### Agent 人格

桌面气泡是唯一聊天入口，始终以宠物身份回应，不跳出人设。

### 自备模型，数据在本机

填入 OpenAI 兼容的 Base URL、Key 与模型名即可对话。密钥与宠物数据默认只存在你的电脑上。PetOS 本身不提供模型服务。

### 选一只属于你的宠物

内置默认伙伴，也可从 Petdex 或本地文件夹导入。每只宠物有自己的形象、性格与记忆空间。

## 开始陪伴

1. **安装** - 在官网下载最新 Windows 或 macOS 安装包  
2. **选宠物** - 使用默认伙伴，或导入你喜欢的形象  
3. **开始相处** - 填入自己的 API，和它说话、陪伴，让它慢慢长大  

官网下载按钮会从 GitHub Releases 拉取对应平台的最新安装包。
